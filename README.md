\## 📖 项目简介 (Overview)

本项目是一套基于 C++ 与 Qt 6 开发的高性能智能安防视频监控管理平台。系统集成了多路视频并发拉流、基于 YOLOv8 的 AI 移动目标检测、智能异常录像与抓拍、历史音视频检索回放以及操作日志安全审计等核心模块。致力于为用户提供一站式、高可靠、现代化的智能安全防护解决方案。



\## ✨ 核心特性 (Key Features)

\* \*\*📡 多路并发监控\*\*：支持 4 路 USB/RTSP 流媒体并发接入，支持单路与四宫格全览视图无缝切换。

\* \*\*🧠 AI 智能侦测 (Producer-Consumer 架构)\*\*：

&#x20;   \* 内置轻量级帧差法前置过滤，结合 YOLOv8 (ONNX Runtime) 进行高精度人形与移动目标检测。

&#x20;   \* 采用 \*\*集中式 AI 异步推断流水线\*\* 与 5 FPS 智能抽频降采样，模型内存零冗余，彻底解决多路并发下的算力抢占与卡顿问题。

\* \*\*📸 异常抓拍与智能录像\*\*：触发预警后自动实施“三连拍”并截取异常视频片段，目标消失后自动安全封包。

\* \*\*🎬 图录无缝联动检索\*\*：

&#x20;   \* 现代化卡片式照片墙图库，支持多维度联合搜索。

&#x20;   \* 行业级\*\*“以图搜录”\*\*交互：在图库双击异常抓拍，系统即刻空降至历史录像中该事件发生的真实时间帧。

\* \*\*⚡ 极速 Thread-Local 数据库连接池\*\*：彻底摒弃传统互斥锁，基于线程本地存储（TLS）设计专属连接隔离池，实现高并发日志与抓拍写入下的绝对无锁与零崩溃。

\* \*\*🎨 现代暗黑极客风 UI\*\*：全面重构 QSS 引擎，实现富文本悬浮窗 (Tooltip)、自适应动态布局与沉浸式黑夜交互体验。



\## 🛠️ 技术栈 (Tech Stack)

\* \*\*核心语言与框架\*\*：C++ 17, Qt 6.11 (MinGW 64-bit), CMake

\* \*\*计算机视觉\*\*：OpenCV 4.x (深度定制编译版，集成 FFmpeg 底层硬解码模块)

\* \*\*深度学习推理\*\*：ONNX Runtime (YOLOv8 预训练模型)

\* \*\*数据持久化\*\*：MySQL 8.0, 纯手工打造无锁线程级连接池



\## 🚀 部署与运行 (Quick Start)

1\. 配置 `bin/db\_config.ini` 中的 MySQL 连接参数。

2\. 导入 `doc/smart\_monitor\_db.sql` 初始化数据库。

3\. 确保 `yolov8n.onnx` 模型文件位于可执行文件同级目录。

4\. 双击 `smart\_monitor.exe` 即可运行。默认管理员账号：`admin` / `123456`。
